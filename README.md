## PROJECT DESCRIPTION:
THIS A WEB BASED CHROME EXTENSION TO TRACK THE PRODUCTIVE AND THE UNPRODUCTIVE USAGE IN THE WEB BROWSER ALONG WITH THIS IT CAN ALSO SHOW
A PIE CHART BASED ANALYSIS AND THE TIME THAT WE STAY ON ANY PARTICULAR SITE. IT SHOWS US THE SUMMARY OF THE USED TIME FOR
THE APPS BE IT PRODUCTIVE OR UNPRODUCTIVE.
